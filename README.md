# 📊 Marketing Campaign Dashboard

## 📌 Overview
This project presents an interactive Power BI dashboard designed to analyze marketing campaign performance across multiple dimensions.  
It helps in understanding user behavior, campaign effectiveness, and supports data-driven decision-making.

---

## 🎯 Objectives
- Analyze campaign performance using key marketing metrics  
- Identify high-performing campaigns and segments  
- Understand customer behavior across different demographics  
- Optimize marketing strategies using data insights  

---

## 🗂️ Data
- Raw marketing campaign dataset  
- Includes impressions, clicks, conversions, CTR, CR, and sales  

---

## 🛠️ Tools Used
- Power BI  
- Excel  

---

## 📊 Key Metrics
- Impressions  
- Clicks  
- Click-Through Rate (CTR)  
- Conversion Rate (CR)  
- Conversions  
- Total Sales  

---

## 🔍 Key Insights
- Campaign performance varies significantly across countries  
- High impressions and clicks do not always lead to high conversions  
- Certain product categories generate higher revenue  
- Customer behavior differs across age groups  

---

## ⚙️ Features
- Interactive filters (Campaign, Country, Age Group, Platform)  
- KPI cards for performance tracking  
- Multi-dimensional analysis (Time, Demographics, Products)  
- Dynamic and user-friendly visualizations
